= 在對象庫存效能頁面上搜尋
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在物件清單效能頁面上搜尋字串。使用位於頁面右上角的「*搜尋*」欄位可以根據物件名稱或策略名稱快速定位資料。這使您能夠快速找到特定物件及其關聯數據，或快速找到策略並查看關聯的策略物件數據。

.步
. 根據您的搜尋要求，執行以下選項之一：
+
|===
| 要找到這個... | 輸入此... 


 a| 
特定對象
 a| 
將物件名稱輸入到*搜尋*欄位中，然後按一下*搜尋*。將顯示您搜尋的對象及其相關資料。



 a| 
使用者定義的效能閾值策略
 a| 
將全部或部分政策名稱輸入到*搜尋*欄位中，然後按一下*搜尋*。將顯示指派給您搜尋的策略的物件。

|===
