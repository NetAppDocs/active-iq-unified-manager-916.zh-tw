= 根據系統定義的效能閾值分析事件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
從系統定義的效能閾值產生的事件表示某個儲存物件的效能計數器或效能計數器集已超過系統定義策略的閾值。這表示儲存物件（例如聚合或節點）正在遇到效能問題。

您可以使用事件詳細資訊頁面來分析效能事件，並在必要時採取糾正措施，使效能恢復正常。

[NOTE]
====
Cloud Volumes ONTAP、 ONTAP Edge 或ONTAP Select系統上未啟用系統定義的閾值策略。

====
