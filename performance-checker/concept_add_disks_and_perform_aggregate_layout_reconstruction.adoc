= 新增磁碟並執行聚合佈局重建
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以向聚合添加磁碟以增加該聚合的儲存容量和效能。新增磁碟後，只有在重建聚合後才能看到效能的提升。

當您在事件詳細資料頁面上收到系統定義的閾值事件時，事件描述文字會列出出現問題的聚合的名稱。您可以在此聚合上新增磁碟並重建資料。

新增到聚合的磁碟必須已存在於叢集中。如果叢集沒有可用的額外磁碟，您可能需要聯絡管理員或購買更多磁碟。您可以使用ONTAP系統管理員或ONTAP命令將磁碟新增至聚合。
