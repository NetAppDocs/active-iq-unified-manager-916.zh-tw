= 編輯使用者定義的效能閾值策略
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以編輯現有效能閾值策略的閾值設定。如果您發現收到的某些閾值條件下的警報過多或過少，這可能會很有用。

.開始之前
您必須具有應用程式管理員角色。

您無法變更現有閾值策略的策略名稱或正在監視的儲存物件的類型。

.步驟
. 在左側導覽窗格中，選擇「*事件閾值*」>「*效能*」。
+
顯示“效能閾值”頁面。

. 選擇要變更的閾值策略，然後按一下「*編輯*」。
+
將顯示「編輯效能閾值策略」頁面。

. 對閾值策略進行更改，然後按一下「*儲存*」。
+
您將返回到“效能閾值”頁面。



儲存後，變更將立即更新至使用該原則的所有儲存物件上。

根據您對策略所做變更的類型，您可能需要查看在警報設定頁面中為使用該策略的物件配置的警報設定。
