= 使用工作負載分析器
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
有多種方法可以從使用者介面啟動工作負載分析器。這裡我們描述如何從左側導覽窗格啟動該工具。

.步驟
. 在左側導覽窗格中，按一下「*工作負載分析*」。
+
將顯示“工作負載分析”頁面。

. 如果您知道工作負載名稱，請輸入該名稱。如果您不確定全名，請輸入至少 3 個字符，系統將顯示與該字串相符的工作負載清單。
. 如果您想查看超過預設 2 小時的統計數據，請選擇時間範圍，然後按一下「*套用*」。
. 查看摘要區域以了解該時間範圍內發生的事件。
. 查看效能和容量圖表，了解任何指標何時出現異常，並查看是否有任何事件與異常條目一致。
