= 審查配置變更對工作負載效能的影響
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
叢集上的配置變更（例如磁碟故障、HA 故障轉移或移動磁碟區）可能會對磁碟區效能產生負面影響並導致延遲增加。

在 Unified Manager 中，您可以查看「工作負載分析」頁面，以了解最近配置變更發生的時間，並將其與操作和延遲（回應時間）進行比較，以查看所選卷工作負載的活動是否發生了變化。

Unified Manager 的效能頁面只能偵測有限數量的變更事件。健康頁面提供由配置變更引起的其他事件的警報。您可以在 Unified Manager 中搜尋該磁碟區以查看事件記錄。
