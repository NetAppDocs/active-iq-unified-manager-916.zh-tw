= 根據使用者定義的效能閾值分析事件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
從使用者定義的閾值產生的事件表示某個儲存物件（例如聚合或磁碟區）的效能計數器已超過您在策略中定義的閾值。這表示集群物件正在遇到效能問題。

您可以使用事件詳細資訊頁面來分析效能事件，並在必要時採取糾正措施，使效能恢復正常。
