= 應用篩選來減少網格中相關物件的列表
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
透過過濾，您可以在網格中顯示更小、定義更明確的物件子集。例如，如果網格中有 25 個卷，則透過過濾，您可以只查看吞吐量小於 90 MBps 或延遲大於 1 ms/op 的捲。
