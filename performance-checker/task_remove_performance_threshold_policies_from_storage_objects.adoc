= 從儲存物件中刪除效能閾值策略
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當您不再希望 Unified Manager 監控效能計數器的值時，您可以從儲存物件中刪除使用者定義的效能閾值策略。

.開始之前
您必須具有應用程式管理員角色。

每次只能從選取物件中刪除一個策略。

您可以透過在清單中選擇多個儲存物件來從多個儲存物件中刪除閾值策略。

.步驟
. 從任何儲存物件的*庫存*頁面中，選擇一個或多個應用了至少一個效能閾值策略的物件。
+
|===
| 清除...的閾值 | 這樣做... 


 a| 
單一對象
 a| 
選取該物件左側的複選框。



 a| 
多個物件
 a| 
選取每個物件左側的複選框。



 a| 
頁面上的所有對象
 a| 
點選image:../media/select_dropdown_65_png.gif["選擇下拉圖示"]在列標題中。

|===
. 按一下“清除效能閾值策略”。
+
將顯示「清除閾值策略」頁面，其中顯示目前指派給儲存物件的閾值策略清單。

. 選擇要從物件中刪除的閾值策略，然後按一下「清除策略」。
+
當您選擇閾值策略時，將顯示該策略的詳細信息，以便您確認已選擇適當的策略。
