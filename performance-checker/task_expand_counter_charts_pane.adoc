= 展開計數器圖表窗格
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以展開計數器圖表窗格，以便圖表更大且更易讀。

定義計數器的比較物件和時間範圍後，您可以查看更大的計數器圖表窗格。您可以使用效能資源管理器視窗中間的 *<* 按鈕來展開窗格。

.步
. 展開或縮小*計數器圖表*窗格。


|===
| 到... | 這樣做... 


 a| 
擴展計數器圖表窗格以適合頁面寬度
 a| 
點擊 *<* 按鈕



 a| 
將計數器圖表窗格縮小到頁面的右半部分
 a| 
點擊 *>* 按鈕

|===
