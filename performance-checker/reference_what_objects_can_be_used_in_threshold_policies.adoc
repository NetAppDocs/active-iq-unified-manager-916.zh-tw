[NOTE]
====
當磁碟區組合策略應用於FlexGroup磁碟區而非FlexVol volume時，只能選擇「Volume IOPS」和「Volume MB/s」屬性作為輔助計數器。如果閾值策略包含節點或聚合屬性之一，則該策略將不會應用於FlexGroup卷，並且您將收到描述這種情況的錯誤訊息。這是因為FlexGroup磁碟區可以存在於多個節點或聚合上。
