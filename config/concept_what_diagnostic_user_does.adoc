= 診斷使用者功能
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
診斷存取的目的是使技術支援能夠幫助您進行故障排除，並且您只能在技術支援的指導下使用它。

診斷使用者可以在技術支援的指導下執行作業系統級命令，以進行故障排除。
