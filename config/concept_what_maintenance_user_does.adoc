= 維護用戶做什麼
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
維護使用者是在 Red Hat Enterprise Linux 系統上安裝 Unified Manager 期間建立的。維護用戶名是“umadmin”用戶。維護使用者在 Web UI 中具有應用程式管理員角色，並且該使用者可以建立後續使用者並為他們指派角色。

維護使用者或 umadmin 使用者也可以存取 Unified Manager 維護控制台。
