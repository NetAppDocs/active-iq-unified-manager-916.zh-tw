= 使用 vSphere VM 控制台存取維護控制台
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果 Unified Manager 使用者介面未執行，或者您需要執行使用者介面中不可用的功能，則可以存取維護控制台來重新設定虛擬設備。

.開始之前
* 您必須是維護使用者。
* 必須開啟虛擬設備才能存取維護控制台。


.步驟
. 在 vSphere Client 中，找到 Unified Manager 虛擬設備。
. 按一下「*控制台*」標籤。
. 按一下控制台視窗內部即可登入。
. 使用您的使用者名稱和密碼登入維護控制台。
+
15 分鐘不活動後，維護控制台將註銷您。
