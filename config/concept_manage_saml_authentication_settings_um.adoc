= 管理 SAML 身份驗證設定
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
設定遠端驗證設定後，您可以啟用安全性斷言標記語言 (SAML) 驗證，以便遠端使用者在存取 Unified Manager Web UI 之前由安全性身分提供者 (IdP) 進行驗證。

請注意，啟用 SAML 身份驗證後，只有遠端使用者才能存取 Unified Manager 圖形使用者介面。本機用戶和維護用戶將無法存取 UI。此配置不會影響存取維護控制台的使用者。
