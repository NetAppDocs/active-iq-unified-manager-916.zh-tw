= 更改本機用戶密碼
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以修改本機使用者登入密碼，以避免潛在的安全風險。

.開始之前
您必須以本機使用者登入。

無法使用這些步驟變更維護使用者和遠端使用者的密碼。若要變更遠端使用者密碼，請聯絡您的密碼管理員。若要變更維護使用者密碼，請參閱link:task_use_maintenance_console.html["使用維護控制台"]。

.步驟
. 登入 Unified Manager。
. 從頂部選單列中，點擊使用者圖標，然後點擊*更改密碼*。
+
如果您是遠端用戶，則不會顯示「*更改密碼*」選項。

. 在「更改密碼」對話方塊中，輸入目前密碼和新密碼。
. 點選“儲存”。


如果 Unified Manager 是在高可用性設定中進行設定的，則必須變更設定的第二個節點上的密碼。兩個實例必須具有相同的密碼。
