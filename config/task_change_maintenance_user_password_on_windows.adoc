= 在 Windows 上變更維護使用者密碼
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以在需要時變更 Unified Manager 維護使用者密碼。

.步驟
. 在 Unified Manager Web UI 登入頁面上，按一下「*忘記密碼*」。
+
將顯示一個頁面，提示您輸入要重設密碼的使用者的姓名。

. 輸入使用者名稱並點擊*提交*。
+
一封包含重設密碼連結的電子郵件將傳送至為該使用者名稱定義的電子郵件地址。

. 點擊電子郵件中的*重設密碼連結*並定義新密碼。
. 返回 Web UI 並使用新密碼登入 Unified Manager。
