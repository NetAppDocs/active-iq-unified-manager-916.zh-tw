= 從註釋中刪除值
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當與自訂註解關聯的值不再適用於該註解時，您可以刪除該值。

.開始之前
* 您必須具有應用程式管理員或儲存管理員角色。
* 註釋值不得與任何註釋規則或群組規則相關聯。


您無法從預定義註解中刪除值。

.步驟
. 在左側導覽窗格中，按一下「儲存管理」>「註解」。
. 在「*註釋*」標籤的註解清單中，選擇要從中刪除值的註解。
. 在「*註釋*」標籤的「*值*」區域中，選擇要刪除的值，然後按一下「*刪除*」。
. 在「*警告*」對話方塊中，按一下「*是*」。
+
該值已被刪除，並且不再顯示在所選註釋的值清單中。
