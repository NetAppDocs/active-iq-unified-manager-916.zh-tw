= 刪除註釋
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當不再需要自訂註釋及其值時，您可以刪除它們。

.開始之前
* 您必須具有應用程式管理員或儲存管理員角色。
* 註釋值不得在其他註釋或群組規則中使用。


.步驟
. 在左側導覽窗格中，按一下「儲存管理」>「註解」。
. 在「註釋」標籤中，選擇要刪除的註釋。
+
顯示所選註釋的詳細資訊。

. 按一下「*動作*」>「*刪除*」以刪除選取的註解及其值。
. 在警告對話方塊中，按一下「*是*」確認刪除。
