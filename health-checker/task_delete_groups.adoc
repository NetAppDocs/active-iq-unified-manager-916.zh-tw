= 刪除群組
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當不再需要某個群組時，您可以從 Unified Manager 中刪除該群組。

.開始之前
* 任何儲存物件（叢集、SVM 或磁碟區）都不能與要刪除的群組關聯的任何群組規則關聯。
* 您必須具有應用程式管理員或儲存管理員角色。


.步驟
. 在左側導覽窗格中，按一下「儲存管理」>「群組」。
. 在「*群組*」標籤中，選擇要刪除的群組，然後按一下「*刪除*」。
. 在「*警告*」對話方塊中，按一下「*是*」確認刪除。
+
刪除群組並不會刪除與該群組關聯的群組操作。但是，刪除群組後，這些群組操作將取消映射。
