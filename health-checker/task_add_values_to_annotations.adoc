= 為註釋添加值
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以為註解新增值，然後將儲存物件與特定的註解名稱-值對關聯。在註釋中新增值可以幫助您更有效地管理儲存物件。

.開始之前
您必須具有應用程式管理員或儲存管理員角色。

您不能為預定義註解新增值。

.步驟
. 在左側導覽窗格中，按一下「儲存管理」>「註解」。
. 在「*註釋*」頁面中，選擇要新增值的註釋，然後按一下「*值*」部分中的「*新增*」。
. 在「新增註釋值」對話方塊中，指定註釋的值。
+
您指定的值對於選定的註釋必須是唯一的。

. 按一下“*新增*”。
