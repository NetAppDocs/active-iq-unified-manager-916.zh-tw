= 查看儲存池詳細信息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以查看儲存池的詳細信息，以監控儲存池的健康狀況、總快取和可用快取以及已使用分配和可用分配。

.開始之前
您必須具有操作員、應用程式管理員或儲存管理員角色。

.步驟
. 在左側導覽窗格中，按一下「*儲存*」>「*聚合*」。
. 按一下聚合名稱。
+
顯示所選聚合的詳細資訊。

. 按一下“磁碟資訊”標籤。
+
顯示詳細的磁碟資訊。

+
[NOTE]
====
僅當選定的聚合正在使用儲存池時才會顯示快取表。

====
. 在快取表中，將指標移到所需儲存池的名稱上。
+
顯示儲存池的詳細資訊。
