= 透過 Web UI 和維護控制台發送支援包
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當您遇到的問題需要比AutoSupport訊息提供的更詳細的診斷和故障排除時，您應該發送支援包。您可以使用 Unified Manager Web Ui 和維護控制台將支援包傳送給技術支援。

Unified Manager 一次最多可儲存兩個完整支援包和三個輕量支援包。
