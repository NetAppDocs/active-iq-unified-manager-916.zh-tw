= 刪除註解規則
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當不再需要註解規則時，您可以從Active IQ Unified Manager中刪除這些規則。

.開始之前
您必須具有應用程式管理員或儲存管理員角色。

當您刪除註釋規則時，註釋將取消關聯並從儲存物件中刪除。

.步驟
. 在左側導覽窗格中，按一下「儲存管理」>「註解」。
. 在「註解規則」標籤中，選擇要刪除的註解規則，然後按一下「刪除」。
. 在「*警告*」對話方塊中，按一下「*是*」確認刪除。
