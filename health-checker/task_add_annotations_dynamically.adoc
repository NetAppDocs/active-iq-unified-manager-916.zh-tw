= 動態新增註釋
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
建立自訂註解時，Unified Manager 會使用規則將叢集、儲存虛擬機器 (SVM) 和磁碟區與註解動態關聯。這些規則會自動將註解指派給儲存物件。

.開始之前
您必須具有應用程式管理員或儲存管理員角色。

.步驟
. 在左側導覽窗格中，按一下「儲存管理」>「註解」。
. 在*註釋*頁面中，按一下*新增註解*。
. 在「新增註解」對話方塊中，鍵入註解的名稱和描述。
. 可選：在*註釋值*部分中，按一下*新增*以向註釋新增值。
. 點選“儲存”。
