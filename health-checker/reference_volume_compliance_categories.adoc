= 容量合規性類別
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
下表描述了 Unified Manager 評估的捲加密參數，以確定磁碟區上的資料是否受到充分保護，以免被未經授權的使用者存取。

請注意，卷加密參數不會影響叢集或儲存虛擬機器是否被視為合規。

[cols="2*"]
|===
| 範圍 | 描述 


 a| 
軟體加密
 a| 
顯示使用NetApp磁碟區加密 (NVE) 或NetApp聚合加密 (NAE) 軟體加密解決方案保護的磁碟區的數量。



 a| 
硬體加密
 a| 
顯示使用NetApp儲存加密 (NSE) 硬體加密保護的捲數。



 a| 
軟體和硬體加密
 a| 
顯示受軟體和硬體加密保護的捲的數量。



 a| 
未加密
 a| 
顯示未加密的捲數。

|===
