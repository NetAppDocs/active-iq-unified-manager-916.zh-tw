= 查看註釋列表和詳細信息
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以查看與叢集、磁碟區和儲存虛擬機器 (SVM) 動態關聯的註釋清單。您還可以查看註釋的詳細信息，例如描述、創建者、建立日期、值、規則以及與註釋相關的物件。

.步驟
. 在左側導覽窗格中，按一下「儲存管理」>「註解」。
. 在「註釋」標籤中，按一下註釋名稱以查看相關詳細資訊。
