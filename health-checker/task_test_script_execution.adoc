= 測試腳本執行
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當儲存物件產生警報時，您可以驗證腳本是否正確執行。

* 您必須具有應用程式管理員或儲存管理員角色。
* 您必須已將支援的文件格式的腳本上傳至 Unified Manager。


.步驟
. 在左側導覽窗格中，按一下「儲存管理」>「腳本」。
. 在腳本頁面中，新增您的測試腳本。
. 在左側導覽窗格中，按一下「儲存管理」>「警報設定」。
. 在「*警報設定*」頁面中，執行下列其中一項操作：
+
[cols="2*"]
|===
| 到... | 這樣做... 


 a| 
添加警報
 a| 
.. 按一下“*新增*”。
.. 在「操作」部分，將警報與您的測試腳本關聯。




 a| 
編輯警報
 a| 
.. 選擇一個警報，然後按一下“*編輯*”。
.. 在「操作」部分，將警報與您的測試腳本關聯。


|===
. 點選“儲存”。
. 在*警報設定*頁面中，選擇您新增或修改的警報，然後按一下*測試*。
+
該腳本使用“-test”參數執行，並將通知警報發送到建立警報時指定的電子郵件地址。
