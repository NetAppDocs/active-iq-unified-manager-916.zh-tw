= 新增安全事件警報
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以為單一安全事件設定警報，就像 Unified Manager 收到的任何其他事件一樣。此外，如果您想平等對待所有安全事件並將電子郵件發送給同一個人，您可以建立單一警報，在觸發任何安全事件時通知您。

.開始之前
您必須具有應用程式管理員或儲存管理員角色。

下面的範例顯示如何為「`Telnet 協定已啟用`」安全事件建立警報。如果配置了 Telnet 存取以對叢集進行遠端管理訪問，則將發送警報。您可以使用相同的方法為所有安全事件建立警報。

.步驟
. 在左側導覽窗格中，按一下「儲存管理」>「警報設定」。
. 在*警報設定*頁面中，按一下*新增*。
. 在“新增警報”對話方塊中，按一下“名稱”，然後輸入警報的名稱和描述。
. 按一下*資源*並選擇要啟用此警報的叢集或叢集。
. 按一下“*事件*”並執行以下操作：
+
.. 在事件嚴重性清單中，選擇*警告*。
.. 在符合事件清單中，選擇*Telnet 協定已啟用*。


. 按一下“*操作*”，然後在“*警告這些使用者*”欄位中選擇將接收警報電子郵件的使用者的姓名。
. 配置此頁面上的任何其他選項，包括通知頻率、發出 SNMP tap 和執行腳本。
. 點選“儲存”。
