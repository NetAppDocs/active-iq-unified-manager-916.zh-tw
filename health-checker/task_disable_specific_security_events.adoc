= 禁用特定安全事件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
預設情況下，所有事件均啟用。您可以停用特定事件以防止產生那些在您的環境中不重要的事件的通知。如果您想要恢復接收已停用的事件的通知，您可以啟用它們。

.開始之前
您必須具有應用程式管理員或儲存管理員角色。

當您停用事件時，系統中先前產生的事件將被標記為過時，並且不會觸發為這些事件配置的警報。當您啟用已停用的事件時，將從下一個監控週期開始產生這些事件的通知。

.步驟
. 在左側導覽窗格中，按一下「*儲存管理*」>「*事件設定*」。
. 在「*事件*設定」頁面中，透過選擇以下選項之一來停用或啟用事件：
+
[cols="2*"]
|===
| 如果你想... | 然後這樣做... 


 a| 
禁用事件
 a| 
.. 按一下“*禁用*”。
.. 在「停用事件」對話方塊中，選擇「*警告*」嚴重性。這是所有安全事件的類別。
.. 在「符合事件」列中，選擇要停用的安全性事件，然後按一下向右箭頭將這些事件移至「停用事件」列。
.. 按一下“儲存並關閉”。
.. 驗證您停用的事件是否顯示在「事件設定」頁面的清單檢視中。




 a| 
啟用事件
 a| 
.. 從已停用事件清單中，選取要重新啟用的一個或多個事件的複選框。
.. 按一下“*啟用*”。


|===
