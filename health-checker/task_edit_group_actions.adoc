= 編輯組操作
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以編輯在 Unified Manager 中設定的群組操作參數，例如群組操作名稱、描述、關聯群組名稱和操作類型的參數。

.開始之前
您必須具有應用程式管理員或儲存管理員角色。

.步驟
. 在左側導覽窗格中，按一下「儲存管理」>「群組」。
. 在「*群組操作*」標籤中，選擇要編輯的群組操作，然後按一下「*編輯*」。
. 在「編輯群組操作」對話方塊中，根據需要變更群組操作名稱、描述、關聯群組名稱和操作類型的參數。
. 點選“儲存”。
