= 在 Red Hat Enterprise Linux 上升級 Union Manager
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當有新版本可用時，您可以升級 Unified Manager。

當NetApp提供 Unified Manager 軟體的修補程式版本時，其安裝流程與新版本相同。

如果 Unified Manager 與OnCommand Workflow Automation實例配對，且這兩種產品都有新版本的軟體可用，則必須中斷這兩種產品的連接，然後在執行升級後建立新的 Workflow Automation 連接。如果您只對其中一個產品進行升級，則應在升級後登入工作流程自動化並驗證它是否仍在從 Unified Manager 取得資料。
