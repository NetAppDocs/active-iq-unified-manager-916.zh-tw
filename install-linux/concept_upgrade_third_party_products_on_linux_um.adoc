= 安裝 Unified Manager 後升級第三方產品
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當 Unified Manager 已安裝在 Linux 系統上時，您可以升級第三方產品，例如 JRE。

開發這些第三方產品的公司會定期報告安全漏洞。您可以按照自己的時間表升級到該軟體的較新版本。
