= 編輯使用者和群組配額電子郵件地址
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以根據與叢集、儲存虛擬機器 (SVM)、磁碟區、qtree、使用者或使用者群組關聯的使用者配額修改電子郵件地址。當您想要覆寫「產生使用者和群組配額電子郵件地址的規則」對話方塊中指定的規則所產生的電子郵件地址時，您可以修改電子郵件地址。

.開始之前
* 您必須具有操作員、應用程式管理員或儲存管理員角色。
* 您必須已經查看過link:reference_rules_to_generate_user_and_group_quota.html["建立規則的指南"]。


如果您編輯電子郵件地址，則產生使用者和群組配額電子郵件地址的規則將不再適用於該配額。若要將通知傳送至指定規則產生的電子郵件地址，您必須刪除該電子郵件地址並儲存變更。

.步驟
. 在左側導覽窗格中，按一下「儲存」>「SVM」。
. 在 *Health: All Storage VMs* 視圖中，選擇一個 SVM，然後按一下 *User and Group Quotas* 選項卡。
. 點擊選項卡行下方的“*編輯電子郵件地址*”。
. 在「編輯電子郵件地址」對話方塊中，執行對應的操作：
+
|===
| 如果... | 然後... 


 a| 
您希望將通知傳送到指定規則產生的電子郵件地址
 a| 
.. 刪除「*電子郵件地址*」欄位中的電子郵件地址。
.. 點選“儲存”。
.. 按 F5 重新整理瀏覽器以重新載入「編輯電子郵件地址」對話方塊。指定規則產生的電子郵件地址顯示在「*電子郵件地址*」欄位中。




 a| 
您希望將通知傳送到指定的電子郵件地址
 a| 
.. 在*電子郵件地址*欄位中修改電子郵件地址。
.. 點選“儲存”。產生使用者和群組配額電子郵件地址的規則不再適用於配額。


|===
