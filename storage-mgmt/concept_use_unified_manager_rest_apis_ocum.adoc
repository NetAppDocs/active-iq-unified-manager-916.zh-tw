= 使用 Unified Manager REST API
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Active IQ Unified Manager為您提供 REST API 來查看有關監控和管理儲存環境的資訊。  API 還允許根據策略配置和管理儲存物件。

您也可以使用 Unified Manager 支援的 API 閘道在所有ONTAP管理的叢集上執行ONTAP API。

有關 Unified Manager REST API 的信息，請參閱link:../api-automation/concept_get_started_with_um_apis.html["Active IQ Unified Manager REST API 入門"]。
