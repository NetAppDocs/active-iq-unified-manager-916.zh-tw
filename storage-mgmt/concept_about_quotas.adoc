= 關於配額
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
配額提供了一種限製或追蹤使用者、群組或 qtree 使用的磁碟空間和檔案數量的方法。您可以使用 `/etc/quotas`文件。配額適用於特定卷或 qtree。
