= 未找到用戶
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當您執行與驗證相關的操作（例如新增、編輯、刪除或測試遠端使用者或群組）時，會顯示下列錯誤訊息： `User not found.`

*原因*

如果使用者存在於 AD 伺服器或 LDAP 伺服器中，並且您將基本可分辨名稱設為不正確的值，則可能會發生此問題。

*糾正措施*

. 在左側導覽窗格中，按一下「*常規*」>「*遠端驗證*」。
. 輸入基本可分辨名稱的適當資訊。
. 點選“儲存”。
