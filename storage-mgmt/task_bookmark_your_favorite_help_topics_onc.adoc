= 為您最喜歡的幫助主題添加書籤
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在「幫助收藏夾」標籤中，您可以為經常使用的幫助主題添加書籤。幫助書籤可讓您快速存取您喜歡的主題。

.步驟
. 導覽至您想要新增為收藏夾的說明主題。
. 按一下“*收藏夾*”，然後按一下“*新增*”。
