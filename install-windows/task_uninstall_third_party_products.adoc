= 卸載第三方產品
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
升級 Unified Manager 後，您可以解除安裝第三方產品，例如 Python 或 Microsoft Visual C++ 2013。這是可選的。



== 解除安裝 Python

.步驟
. 從*控制台*中選擇*程式* > *程式和功能*。
. 從程式列表中選擇 Python。
. 選擇*卸載此程式*。卸載除 3.12 之外的任何版本的 Python。確保保留 Python 版本 3.12。




== 解除安裝 Microsoft Visual C++ 2013

.步驟
. 從*控制台*中選擇*程式* > *程式和功能*。
. 從程式清單中選擇 *Microsoft Visual C++ 2013*。
. 選擇*卸載此程式*。
