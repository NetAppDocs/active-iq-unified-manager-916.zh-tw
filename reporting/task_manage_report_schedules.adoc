= 管理報告時間表
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以從「報告計畫」頁面管理您的報告計畫。您可以查看、修改或刪除現有計劃。

[NOTE]
.開始之前
====
您無法從「報告計畫」頁面安排新報告。您只能從物件庫存頁面新增計劃報告。

====
* 您必須具有應用程式管理員或儲存管理員角色。


.步驟
. 在左側導覽窗格中，按一下「*儲存管理*」>「*報告計畫*」。
. 在「報告計畫」頁面上：
+
[cols="2*"]
|===
| 如果你想... | 然後... 


 a| 
查看現有時間表
 a| 
使用捲軸和頁面控制項捲動現有報告清單。



 a| 
編輯現有時間表
 a| 
.. 點擊更多圖標image:../media/more_icon.gif[""]您想要使用的時間表。
.. 按一下“*編輯*”。
.. 做出必要的改變。
.. 按一下複選標記以儲存變更。




 a| 
刪除現有計劃
 a| 
.. 點擊更多圖標image:../media/more_icon.gif[""]您想要使用的時間表。
.. 按一下“*刪除*”。
.. 確認你的決定。


|===
