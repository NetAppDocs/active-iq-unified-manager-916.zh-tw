= 自訂儲存虛擬機器報告
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以建立儲存虛擬機器報告來分析磁碟區資訊並查看整體運作狀況和儲存可用性。例如，您可以建立報表來顯示達到最大磁碟區數的 SVM 並分析已停止的 SVM。
