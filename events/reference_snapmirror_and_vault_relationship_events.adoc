= 非同步鏡像和 Vault 關係事件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
非同步鏡像和 Vault 關係事件為您提供有關非同步SnapMirror和 Vault 關係狀態的信息，以便您可以監控潛在問題。捲和儲存虛擬機器保護關係均支援非同步鏡像和保險庫關係事件。但儲存虛擬機器災難復原僅不支援 Vault 關係。事件按影響區域分組，包括事件和陷阱名稱、影響等級、來源類型和嚴重性。
