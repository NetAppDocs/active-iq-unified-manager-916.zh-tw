= 啟用和停用已解決和已過時事件的警報
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
對於您已配置為發送警報的所有事件，當這些事件轉變至所有可用狀態時都會發送警報訊息：新建、已確認、已解決和過時。如果您不想在事件進入「已解決」和「過時」狀態時收到警報，您可以設定全域設定來抑制這些警報。

.開始之前
您必須具有應用程式管理員或儲存管理員角色。

預設情況下，當事件進入「已解決」和「過時」狀態時，不會發送警報。

.步驟
. 在左側導覽窗格中，按一下「儲存管理」>「警報設定」。
. 在「*警報設定*」頁面中，使用「已解決和過時事件的警報」項目旁的滑桿控制執行下列其中一項操作：
+
|===
| 到... | 這樣做... 


 a| 
當事件解決或過時時停止發送警報
 a| 
將滑桿控制項向左移動



 a| 
當事件解決或過時時開始發送警報
 a| 
將滑桿控制項向右移動

|===
