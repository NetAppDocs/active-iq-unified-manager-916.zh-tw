= 確認並解決事件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您應該在開始處理產生事件的問題之前確認該事件，這樣您就不會繼續收到重複的警報通知。對特定事件採取糾正措施後，您應該將該事件標記為已解決。

.開始之前
您必須具有操作員、應用程式管理員或儲存管理員角色。

您可以同時確認並解決多個事件。

[NOTE]
====
您無法確認訊息事件。

====
.步驟
. 在左側導覽窗格中，按一下「事件管理」。
. 從事件清單中，執行以下操作來確認事件：
+
|===
| 如果你想... | 這樣做... 


 a| 
確認並將單一事件標記為已解決
 a| 
.. 按一下事件名稱。
.. 從事件詳細資訊頁面確定事件的原因。
.. 點選*確認*。
.. 採取適當的糾正措施。
.. 按一下“標記為已解決”。




 a| 
確認多個事件並將其標記為已解決
 a| 
.. 從相應的事件詳細資訊頁面確定事件的原因。
.. 選擇事件。
.. 點選*確認*。
.. 採取適當的糾正措施。
.. 按一下“標記為已解決”。


|===
+
事件被標記為已解決後，該事件將被移至已解決事件清單。

. *可選*：在*註釋和更新*區域，新增有關如何處理事件的註釋，然後按一下*發布*。
