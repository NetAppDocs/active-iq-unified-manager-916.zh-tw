= 刪除警報
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當不再需要警報時，您可以刪除它。例如，當某個資源不再受 Unified Manager 監控時，您可以刪除為該資源建立的警報。

.開始之前
您必須具有應用程式管理員角色。

.步驟
. 在左側導覽窗格中，按一下「儲存管理」>「警報設定」。
. 在*警報設定*頁面上，選擇要刪除的警報，然後按一下*刪除*。
. 按一下“是”確認刪除請求。
