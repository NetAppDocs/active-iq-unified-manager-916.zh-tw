= 查看未指派的事件
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以查看未指派的事件，然後將每個事件指派給可以解決它們的使用者。

.開始之前
您必須具有操作員、應用程式管理員或儲存管理員角色。

.步驟
. 在左側導覽窗格中，按一下「事件管理」。
+
預設情況下，新事件和已確認事件顯示在事件管理庫存頁面上。

. 從「*篩選器*」窗格中，選擇「*指派給*」區域中的「*未指派*」篩選器選項。
