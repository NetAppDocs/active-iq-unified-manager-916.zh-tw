= 如何計算物件狀態
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
物件狀態由目前處於「新」或「已確認」狀態的最嚴重事件決定。例如，如果物件狀態為“錯誤”，則該物件事件之一的嚴重性類型為“錯誤”。採取糾正措施後，事件狀態將變為「已解決」。
