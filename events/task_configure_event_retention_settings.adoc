= 配置事件保留設定
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以指定事件在自動刪除之前在 Unified Manager 伺服器中保留的月份數。

.開始之前
您必須具有應用程式管理員角色。

保留事件超過 6 個月可能會影響伺服器效能，因此不建議這樣做。

.步驟
. 在左側導覽窗格中，按一下「*常規*」>「*資料保留*」。
. 在*資料保留*頁面中，選擇事件保留區域中的滑桿並將其移至應保留事件的月份數，然後按一下*儲存*。
