= 更改或取消計劃的維護時段
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您已設定將來發生的 Unified Manager 維護窗口，則可以變更開始時間和結束時間或取消發生維護窗口。

.開始之前
您必須具有應用程式管理員或儲存管理員角色。

如果您在規劃的維護時段結束時間之前完成了叢集維護，並且想要再次開始接收來自叢集的事件和警報，則取消目前正在執行的維護時段很有用。

.步驟
. 在左側導覽窗格中，按一下「儲存管理」>「叢集設定」。
. 在叢集的「維護模式」欄位中：
+
|===
| 如果你想... | 執行此步驟... 


 a| 
更改計畫維護時段的時間範圍
 a| 
.. 點擊滑桿按鈕旁邊的文字“Scheduled”。
.. 變更開始和/或結束日期和時間，然後按一下*套用*。




 a| 
延長活動維護視窗的長度
 a| 
.. 點擊滑桿按鈕旁邊的文字“Active”。
.. 變更結束日期和時間，然後按一下「*套用*」。




 a| 
取消計畫的維護時段
 a| 
選擇滑桿按鈕並將其向左移動。



 a| 
取消活動維護時段
 a| 
選擇滑桿按鈕並將其向左移動。

|===
