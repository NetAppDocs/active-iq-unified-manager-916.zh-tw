= 從卷/健康詳情頁面恢復已暫停關係的預定傳輸
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
在您停止關係以停止發生計劃的傳輸後，您可以使用磁碟區/健康詳細資料頁面上的*還原*重新啟用計劃的傳輸，以便保護來源或主磁碟區上的資料。如果存在檢查點，則在下一個預定的傳輸間隔從檢查點恢復傳輸。

.開始之前
* 您必須具有應用程式管理員或儲存管理員角色。
* 您必須已經設定了工作流程自動化。


.步驟
. 在「*Volume / Health*」詳細資訊頁面的「*Protection*」標籤中，在拓樸視圖中右鍵點選要恢復的靜止關係。
+
或者，從 *操作* > *關係* 選單中選擇 *恢復*。

. 在「*恢復*」對話方塊中，按一下「*繼續*」。
+
您將返回「音量/健康」詳細資訊頁面。

. 若要查看相關的工作任務並追蹤其進度，請按一下「*Volume / Health*」詳細資料頁面頂部顯示的作業連結。
. 在「*作業*」詳細資料頁面中，按一下「*重新整理*」以更新與保護設定作業相關的任務列表和任務詳細信息，並確定作業何時完成。
+
作業完成後，資料傳輸將在下一個預定的傳輸間隔恢復。
