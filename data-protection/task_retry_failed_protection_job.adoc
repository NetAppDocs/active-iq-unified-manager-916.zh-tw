= 重試失敗的保護作業
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
採取措施修復失敗的保護作業後，您可以使用*重試*再次執行該作業。重試作業將使用原始作業 ID 建立新作業。

.開始之前
您必須具有應用程式管理員或儲存管理員角色。

每次只能重試一項失敗的作業。選擇多個作業將停用「重試」按鈕。僅可重試保護配置和保護關係操作類型的作業。

.步驟
. 在左側導覽窗格中，按一下「*保護*」>「*作業*」。
. 從作業清單中，選擇單一失敗的保護配置或保護關係操作類型作業。
+
*重試*按鈕已啟用。

. 按一下“*重試*”。
+
工作重新開始。
