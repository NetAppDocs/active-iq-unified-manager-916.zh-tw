= 監控作業
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以使用「作業」頁面來監視作業狀態並查看作業屬性（例如儲存服務類型、狀態、提交時間和完成時間），以確定作業是否已成功完成。

.開始之前
您必須具有應用程式管理員或儲存管理員角色。

.步驟
. 在左側導覽窗格中，按一下「*保護*」>「*作業*」。
+
將顯示「作業」頁面。

. 查看*狀態*列以確定目前正在執行的作業的狀態。
. 點擊工作名稱即可查看有關該特定工作的詳細資訊。
+
將顯示「作業詳細資料」頁面。
