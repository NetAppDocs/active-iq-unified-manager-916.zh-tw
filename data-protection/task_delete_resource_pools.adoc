= 刪除資源池
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
當不再需要資源池時，您可以刪除它。例如，您可能想要將成員聚合從一個資源池重新指派到其他幾個資源池，從而使原始資源池變得過時。

.開始之前
您必須具有應用程式管理員或儲存管理員角色。

僅當選擇至少一個資源池時，「*刪除*」按鈕才可用。

.步驟
. 在左側導覽窗格中，按一下「*保護*」>「*資源池*」。
. 選擇要刪除的資源池。
. 按一下“*刪除*”。
+
此資源池從資源池清單中刪除，其聚合也從成員清單中刪除。
